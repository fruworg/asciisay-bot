Вариация cowsay, сделанная телеграм-ботом. В арсенале 33 животного.
Животного можно выбрать самому или довериться случаю.

Сделано при помощи:
[asciiart.eu](https://www.asciiart.eu/animals/) + [tbot](https://github.com/yanzay/tbot) <br />
